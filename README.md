## Commands
### Player Commands
* ```/arcana version``` Shows the current Arcana Novum mod version.
* ```/arcana help``` Opens the Arcana Guide Book GUI.
* ```/arcana guide``` Opens the Arcana Guide Book GUI (alias for help).
* ```/arcana items``` Shows your current Arcane Concentration usage and a breakdown of items taking concentration.
* ```/arcana show``` Shows off the Arcana Item in your main hand to all players on the server.
* ```/arcana blocks``` Lists all of your placed Arcana Blocks, including their location and dimension.

### Admin Commands
* ```/arcana create <id> [<targets>]``` Creates an Arcana Item by its ID and gives it to yourself, or to the specified target players.
* ```/arcana cache``` Opens the Compendium GUI for browsing and retrieving Arcana Items and ingredients.
* ```/arcana reload``` Reloads the Arcana Novum configuration and refreshes recipes.
* ```/arcana blocks <player>``` Lists all of a specified player's placed Arcana Blocks.
* ```/arcana uuids <player>``` Lists all Arcana Items in a player's inventory with their unique IDs, and flags any duplicates.
* ```/arcana infuse <percentage> [<target>]``` Infuses the held item to the specified enhancement percentage (0-100). Use 0 to strip enhancements.
* ```/arcana xp add <targets> <amount> [points|levels]``` Adds Arcana XP points or levels to the selected players.
* ```/arcana xp set <targets> <amount> [points|levels]``` Sets the Arcana XP points or levels for the selected players.
* ```/arcana xp query <target>``` Queries a player's current Arcana Level and XP.
* ```/arcana skillpoints add <targets> <amount>``` Adds skill points to the selected players.
* ```/arcana skillpoints set <targets> <amount>``` Sets the skill points for the selected players.
* ```/arcana skillpoints query <target>``` Queries a player's current skill points.
* ```/arcana achievement grant <id> <targets>``` Grants an achievement to the selected players.
* ```/arcana achievement revoke <id> <targets>``` Revokes an achievement from the selected players.
* ```/arcana achievement query <id> <target>``` Queries whether a player has a specific achievement.
* ```/arcana research grant <id> <targets>``` Grants research for an Arcana Item to the selected players.
* ```/arcana research revoke <id> <targets>``` Revokes research for an Arcana Item from the selected players.
* ```/arcana research query <id> <target>``` Queries whether a player has researched a specific Arcana Item.
* ```/arcana augment apply <id> <level> [<target>]``` Applies an Augment at the specified level to the held Arcana Item. Cannot downgrade existing augments.
* ```/arcana augment setlevel <id> <level> [<target>]``` Sets an Augment to the specified level for a player. Use level 0 to remove.
* ```/arcana changeCrafter <username> crafted|synthesized|earned|found``` Changes the crafter tag on the held Arcana Item to the specified player and acquisition type.
* ```/arcana changeSkin <skin>``` Changes the cosmetic skin on the held Arcana Item to an unlocked skin, or "none" to reset to default.
* ```/arcana boss start dragon``` Starts the Dragon Boss fight.
* ```/arcana boss resetAbilities <doAbility>``` Resets the boss's ability cycle. Set doAbility to true to immediately trigger a new ability.
* ```/arcana boss forceLairAction``` Forces the boss to perform a lair action.
* ```/arcana boss abort``` Aborts the current boss fight.
* ```/arcana boss clean``` Cleans up any lingering entities or effects from a boss fight.
* ```/arcana boss status``` Displays the current status of the active boss fight.
* ```/arcana boss forcePlayerCount <players>``` Overrides the player count used for boss difficulty scaling.
* ```/arcana boss announce <time>``` Announces an upcoming boss fight to all players with a specified time message.
* ```/arcana boss begin``` Begins the boss fight encounter.
* ```/arcana boss teleport <player>``` Teleports a specific player to the boss fight arena.
* ```/arcana boss teleport all``` Teleports all players to the boss fight arena.
* ```/arcana specialEvent check <player>``` Checks whether a player meets the conditions for a special event.
* ```/arcana specialEvent trigger <players> gaialtus``` Triggers the Gaialtus special event for the selected players.
* ```/arcana specialEvent trigger <players> ceptyus``` Triggers the Ceptyus special event for the selected players.
* ```/arcana specialEvent trigger <players> zeraiya``` Triggers the Zeraiya special event for the selected players.

### Admin Configuration Commands
All configuration values can be viewed and changed with `/arcana config <setting> [value]`. Omitting the value will display the current setting. Changes take effect immediately and are persisted across restarts.

#### General Settings
* `/arcana config recipeFolder` The subfolder that is used to currently load recipes for Arcana. (default: "default")
* `/arcana config doConcentrationDamage` Whether players are damaged for going over their concentration limit. (default: true)
* `/arcana config announceAchievements` Whether it is announced in chat when players complete achievements, reach levels, or craft new items. (default: true)
* `/arcana config researchEnabled` When enabled, items must be researched in order to be crafted. (default: true)
* `/arcana config stardustParticleRate` The chance every 3 ticks that Stardust Infused gear emits visible particles to others when worn/held. Set to 0 to disable. (default: 0.225)
* `/arcana config disableArcanaCrafting` If true, players will not be able to craft their own Arcana items. (default: false)
* `/arcana config disableStardustInfusion` If true, players will not be able to perform the stardust infusion minigame for their gear. They will still be able to use it for making Sovereign Paper. (default: false)
* `/arcana config logCommandUsage` Whether command usage is logged to the server console. (default: false)
* `/arcana config allowSimilarBlockChecks` When true, blocks sharing a name but from a different mod can be used in multiblocks or for crafting the Starlight Forge or Midnight Enchanter. This is needed for mod compatibility for mods that alter or remove vanilla blocks. (default: true)

#### Stardust Infusion Maximums
* `/arcana config infusionMaxDurability` The maximum durability multiplier achievable through stardust infusion. (default: 0.5)
* `/arcana config infusionMaxMiningSpeed` The maximum mining speed multiplier achievable through stardust infusion. (default: 1.25)
* `/arcana config infusionMaxAttackSpeed` The maximum attack speed bonus achievable through stardust infusion. (default: 0.5)
* `/arcana config infusionMaxAttackDamage` The maximum attack damage bonus achievable through stardust infusion. (default: 5.0)
* `/arcana config infusionMaxArmor` The maximum armor bonus achievable through stardust infusion. (default: 5.0)
* `/arcana config infusionMaxArmorToughness` The maximum armor toughness bonus achievable through stardust infusion. (default: 5.0)
* `/arcana config infusionMaxKnockbackResistance` The maximum knockback resistance bonus achievable through stardust infusion. (default: 0.15)
* `/arcana config infusionMaxHealthBoost` The maximum health boost bonus achievable through stardust infusion. (default: 5.0)

#### Runic Arcane Flak Arrows
